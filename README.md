# RPNCalculator
Reverse Polish Notation Calculator

It was somewhat difficult to get started on this project as I had no idea what a reverse polish notation calculator was. A Reverse Polish Notation (RPM) calculator essentially takes the values in first, and then applies the respective operand. Once understanding that, I wrote the header file to declare basic operations, so I could have an idea of what I was implementing. I worked on the implementation file so the header was usable. After that, I finally wrote out the logic and used a stack underneath to keep track of values. Finally, I wrote more complex tests to see if my logic was sound, and after debugging it finally was.
